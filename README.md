# Power BI Data Analysis Projects

## Data Professional Survey Breakdown
![Data Professional Survey Breakdown](BI_Salary_Study_Interractive_Dashboard.png)

This project analyzes survey data from data professionals using Power BI to uncover insights into salary trends, job roles, tool usage, and career satisfaction.

An interactive dashboard was created to visualize key metrics such as average salary by role, tool popularity, job satisfaction levels, and country-wise distribution. The project demonstrates strong skills in data cleaning, DAX calculations, dashboard design, and business-oriented data storytelling.
